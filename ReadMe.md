Java Project Collection

Overview

This project contains two standalone Java applications designed to demonstrate basic object-oriented programming and console-based management systems.

Files
1. GiftCardSystem.java
A simplified system for managing digital gift cards and customer associations.

Description: This program simulates the lifecycle of a gift card, including creation, balance top-ups, and transactions.

Key Classes:

Customer: Stores customer details (Name, Email).

GiftCard: Manages card data (ID, Balance, Owner) and operations (addBalance, useCard).

Functionality: The main method demonstrates creating a customer, initializing a card with a balance, adding funds, and making a purchase.

2. StudentManagement.java
A menu-driven console application for managing student records.

Description: allows users to interactively manage a small database of students using standard input/output.

Key Features:

Add Student: Input ID, Name, and Marks to store a new student record (limit: 5 students).

Display All: View a list of all currently stored students.

Search: Find specific student details using their unique ID.

Interaction: The program runs in a loop, presenting a menu until the user selects the "Exit" option.
